# 项目布局更新说明

## 更新内容

### 1. 新增主布局组件 (MainLayout)

创建了全新的主布局组件 `src/components/layout/MainLayout.tsx`，包含：

#### 功能特性：
- **侧边栏导航**：可折叠的侧边栏菜单，支持响应式设计
- **顶部页头**：包含菜单折叠按钮、通知图标、用户信息下拉菜单
- **菜单项**：
  - 首页
  - 问卷管理
  - 答卷管理
  - 数据分析

#### 设计亮点：
- 深色主题侧边栏，现代化设计
- 菜单项带图标和悬停效果
- Logo 区域展示系统名称
- 响应式布局，适配移动端
- 平滑的过渡动画

### 2. 新增首页 Dashboard

创建了全新的首页 `src/pages/home/index.tsx`，包含：

#### 页面模块：
- **欢迎横幅**：渐变色背景的欢迎区域
- **数据统计卡片**：显示问卷总数、答卷总数、今日答卷、进行中问卷
- **快捷操作区**：创建问卷、问卷列表、数据分析的快速入口
- **功能介绍**：系统主要功能模块展示
- **使用提示**：帮助用户快速上手

#### 特点：
- 响应式网格布局
- 卡片悬停动效
- 清晰的视觉层次
- 便捷的快速操作入口

### 3. 路由系统优化

更新了路由配置 `src/router/router.tsx`：

- 登录页面独立，不使用主布局
- 其他所有页面统一使用主布局包裹
- 默认路由重定向到首页
- 支持精确路由匹配

### 4. 样式优化

#### MainLayout 样式：
- 侧边栏：220px 宽度，可折叠
- 页头：64px 高度，白色背景，阴影效果
- 内容区：自适应高度，浅灰色背景
- 响应式设计：移动端优化

#### Home 页面样式：
- 卡片布局，统一间距
- 悬停动画效果
- 响应式网格系统
- 移动端适配

## 使用方式

### 主布局使用

所有需要侧边栏和页头的页面会自动被 `MainLayout` 包裹，无需手动导入。

### 编辑页面布局

编辑类页面（如问卷编辑）仍然使用 `BaseLayout`：

```tsx
import BaseLayout from '@/components/layout/BaseLayout'

const EditPage = () => {
  return (
    <BaseLayout 
      header="页面标题"
      footerButtons={['break', 'saveToNext']}
      submitFn={handleSubmit}
    >
      {/* 页面内容 */}
    </BaseLayout>
  )
}
```

### 普通页面布局

列表和展示类页面直接返回内容即可：

```tsx
const ListPage = () => {
  return (
    <div style={{ padding: '24px' }}>
      {/* 页面内容 */}
    </div>
  )
}
```

## 技术栈

- **UI 框架**：Ant Design 4.16+
- **布局组件**：Layout, Menu, Dropdown, Avatar, Badge
- **图标**：@ant-design/icons
- **路由**：React Router DOM 5.x
- **样式**：SASS
